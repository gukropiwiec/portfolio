## Run this project locally

If you want to run this project locally, you can follow the step-by-step:

### Prerequisites

-   npm
-   git
-   Angular CLI
    ```sh
    npm install -g @angular/cli
    ```

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/gukropiwiec/portfolio.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Launch the server
    ```sh
    npm start
    ```
